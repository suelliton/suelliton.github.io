About Drumdiy Controller

The project

The Drumdiy Controller came from your developer's dissatisfaction with the free tools available. The lack of support and abandonment on the part of the developers contributed to the precariousness in some situations, confusing the users, and giving headaches to those who only want to play their drum diy.

In order to solve this situation, the present software was developed with the newest technologies, some ideas of the original software were preserved and others improved, besides additions of new functionalities aiming a complete and intuitive operation

Features

Program the Arduino

Send the sketch to your Arduino Mega 2560 with one click.

Channel-based configuration

Addition of N channels with various configurations.

LoopMIDI

Automatically check, install, and execute the LoopMIDI. No need to run the loopMIDI manually.

Channel controls

Gain, Threshold, Mask, Scan, Retrigger, Note and Pin, being able to address any pin to any note.

Manipulate channels at run time

Add, edit, remove, and mute.

Set up and save

Send setup to Arduino with one click, test and save when you're perfect.The configuration will be saved in a JSON file.

MIDI and Serial ports

Identify MIDI and Serial Ports at run time.

Windows installer

The installer makes it easy to install the software and creates a link on the desktop.

Microdrum

Compatible with Arduino microdrum code for config tools.

Possible problems

Operating systems with opengl support lower than version 2.0 will present an error like the one shown below.

OpenGL version detected: 1.1
Try upgrading your graphics drivers and/or your graphics hardware in case of problems.
The application will leave now.

The solution to this is to upgrade the opengl version. The updated file can be found here

After downloading the file should be unzipped and the openGL.dll file should be copied to the C: \ Windows \ System32 folder. If there are problems with folder permissions, use the following tutorial.
