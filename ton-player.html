About Ton Player

The project

Ton Player came from the dissatisfaction of its developer with the free tools of local audio players available on linux and windows systems. Problems with functionality, bugs involving existing players and the difficulty of keeping files organized, metadata and playlists were the main motivators since the project.

This software was developed with the newest application building frameworks, interconnects Spotify and Shazam APIs for metadata and file searches in addition to the SpotDL project for downloading music and other auxiliary libraries for metadata management of mp3 files.

Features

Download and play musics

Use Ton Player for download musics direct from youtube.

Create Playlists

Playlists can be added from local files or downloaded files, manual or automatically..

Load and Play local MP3 files

Load any MP3 local file and your metadata if exists.

Download metadata of musics

You can download the metadata using the recognize of Shazam API.

Write metadata music

Ton Player download the metadata in your database and you can write this for local MP3 files.

Search any album

Search from Spotify and download from YouTube.

Management

Organize your musics and playlists like you want.

No ADS

No ADS, no interruptions.

Step by step

  1. Download the .deb file from Ton-Player-1.3.deb
  2. Download the setup_linux.sh file from setup_linux.sh
  3. install clicking in the file or with the command:

    $ sudo dpkg -i Ton-Player-1.3.deb

  4. open the terminal and enter the directory where you downloaded setup_linux.sh, normally:

    $ cd ~USERNAME/Downloads

  5. Give permission to run setup_linux.sh:

    $ sudo chmod +x setup_linux.sh

  6. Run the setup_linux.sh:

    $ sudo ./setup_linux.sh

  7. Run the first time using terminal to see logs and identify possible problems:

    $ ton-player

  8. Search the Ton Player in app menu
  9. Enjoy!!!
